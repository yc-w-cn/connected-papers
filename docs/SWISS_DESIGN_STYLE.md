# Swiss 设计风格指南

## 核心原则

### 1. 网格系统
- 使用严格的网格布局作为设计基础
- 所有元素对齐到网格线
- 保持一致的间距和比例
- 使用黄金比例或斐波那契数列确定网格尺寸

### 2. 字体选择
- 优先使用无衬线字体（Helvetica, Arial, Inter 等）
- 字体层级清晰：标题、副标题、正文、辅助文本
- 使用粗细对比建立视觉层次
- 保持字体大小的一致性（建议使用 8pt 的倍数）

### 3. 排版原则
- 左对齐为主，避免居中对齐
- 行高设置为字体大小的 1.5-2 倍
- 段落间距保持一致
- 使用字间距和词间距微调可读性

### 4. 色彩系统
- 使用有限的调色板（建议 3-5 种主色）
- 高对比度配色方案
- 纯色为主，避免渐变
- 黑白灰作为基础色调，辅以 1-2 种强调色

### 5. 留白与空间
- 大量使用留白创造呼吸感
- 元素之间保持充足间距
- 避免过度填充空间
- 使用负空间引导视线流动

### 6. 几何形状
- 使用基本几何形状（圆形、方形、三角形）
- 保持形状的简洁和纯粹
- 避免装饰性元素
- 形状与功能紧密结合

### 7. 视觉层次
- 通过大小、粗细、颜色建立清晰的层次
- 重要信息突出显示
- 次要信息适当弱化
- 保持视觉路径的流畅性

### 8. 图形元素
- 使用简洁的图标和插图
- 避免复杂的装饰性图形
- 图形与文字保持风格统一
- 优先使用摄影而非插画

### 9. 布局结构
- 不对称布局增加动态感
- 保持视觉平衡
- 使用对角线和斜线创造动感
- 元素排列有节奏感

### 10. 功能性优先
- 设计服务于功能
- 避免不必要的装饰
- 清晰的信息传达
- 优秀的用户体验

## 技术实现建议

### CSS 变量示例
```css
:root {
  /* 字体 */
  --font-family: 'Inter', 'Helvetica Neue', Helvetica, Arial, sans-serif;
  
  /* 字体大小（8pt 倍数） */
  --text-xs: 12px;
  --text-sm: 16px;
  --text-base: 24px;
  --text-lg: 32px;
  --text-xl: 48px;
  --text-2xl: 64px;
  
  /* 间距（8px 倍数） */
  --space-1: 8px;
  --space-2: 16px;
  --space-3: 24px;
  --space-4: 32px;
  --space-5: 48px;
  --space-6: 64px;
  
  /* 颜色 */
  --color-primary: #000000;
  --color-secondary: #333333;
  --color-accent: #FF3B30;
  --color-background: #FFFFFF;
  --color-surface: #F5F5F5;
  
  /* 行高 */
  --line-height-tight: 1.5;
  --line-height-normal: 1.75;
  --line-height-loose: 2;
}
```

### 网格系统示例
```css
.container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: var(--space-3);
  padding: var(--space-4);
  max-width: 1200px;
  margin: 0 auto;
}
```

## 常见错误避免

- ❌ 使用过多装饰性元素
- ❌ 过度使用渐变和阴影
- ❌ 不一致的间距和对齐
- ❌ 过多的颜色和字体
- ❌ 缺乏视觉层次
- ❌ 过度拥挤的布局
- ❌ 使用衬线字体作为主要字体
