# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-01-02

### 新增
- 实现学术文献关系网络可视化功能
- 添加 D3.js 依赖和数据导出功能
- 添加加载进度条组件并改进页面加载状态处理
- 添加节点平均引用计算和悬停提示功能
- 在页脚添加版本号和构建日期信息
- 添加论文处理和引用获取脚本
- 添加网络请求记录和引用文献管理功能
- 添加批量获取论文引用文献功能并优化单篇获取逻辑
- 为 API 请求添加请求头和响应体记录功能
- 添加 Jest 单元测试配置和多个模块的单元测试

## [0.0.1] - 2025-11-17
