# 更新日志

本项目遵循[语义化版本](https://semver.org/lang/zh-CN/)。

## [1.1.1] - 2026-01-02

### 修复
- 修复 LoadingStatCards 组件中 animationRef 类型定义不完整的问题

## [1.1.0] - 2026-01-02

### 新增
- 实现数据分片加载功能以提升大文件加载体验
- 添加网络图节点类型切换功能，支持动态切换显示的节点类型
- 添加学术网络数据文件及清单

### 改进
- 优化加载动画，使用 requestAnimationFrame 实现平滑动画效果
- 添加加载过程中的统计卡片动画展示
- 优化力导向图实现，提升可视化性能
- 使用 cn 工具函数优化类名合并，提高代码可读性

### 修复
- 修正数据文件路径以匹配部署环境

### 重构
- 简化链接类型定义
- 优化动画引用类型并添加时间戳参数
- 移除未使用的 currentChunk 属性
- 更新 html 语言设置为 zh-CN

### 清理
- 清理不再需要的 .gitignore 条目和 data.json 文件

## [1.0.0] - 2026-01-02

### 新增
- 实现学术文献关系网络可视化功能
- 添加 D3.js 依赖和数据导出功能
- 添加加载进度条组件并改进页面加载状态处理
- 添加节点平均引用计算和悬停提示功能
- 在页脚添加版本号和构建日期信息
- 添加论文处理和引用获取脚本
- 添加网络请求记录和引用文献管理功能
- 添加批量获取论文引用文献功能并优化单篇获取逻辑
- 为 API 请求添加请求头和响应体记录功能
- 添加 Jest 单元测试配置和多个模块的单元测试
